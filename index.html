4 个精选工作流

常见的品牌营销 Skill 组合玩法。

每个都列出了用到哪些 Skill。直接照抄，或者换几个 Skill 改成你自己的。

自己组合一个工作流

上面 4 个不够用？
自己组一个。

不需要懂技术，4 步就能搞定。

01
写下你想让 AI 帮你做的事
举例 "每周一次，给我品牌的小红书账号产出 5 条新笔记。"
02
拆成几个步骤 → 每步找对应的 Skill
① 扒一周同赛道爆款 → 🔍 扒信息
② 提炼 5 个选题角度 → ✍️ 写内容
③ 写 5 条图文 → ✍️ 写内容
④ 去 AI 味 → ✍️ 写内容（humanizer-zh）
⑤ 配图 → 🎨 做视觉
⑥ 自动发布 → 🤖 跑自动化
03
在对应分类里挑 Skill
挑出来 competitor-profiling · content-planner · copywriting · humanizer-zh · marketing-materials · xiaohongshu-mcp
从「Skill 列表」点开看安装命令，全部复制粘贴装好。
04
粘进 AI，一句话让它全部跑完
请按以下步骤帮我做"周更小红书"： 1. 用 competitor-profiling 扒"心理疗愈"赛道近一周 top 30 爆款 2. 用 content-planner 给我 5 个选题角度 3. 用 copywriting 写 5 条图文（标题 + 正文 + tag） 4. 用 humanizer-zh 全部过一遍去 AI 味 5. 用 marketing-materials 给我配图 prompt 6. 用 xiaohongshu-mcp 定时今晚 8 点发 请一口气跑完，不要每步等我。
不需要懂原理，装上就能用。

装 AI Agent → 复制 → 粘贴 → 用。

不需要会写代码。装一个 AI Agent，从这个页面复制安装命令粘进去，AI 会自己装好。

01
装一个 AI Agent

国内外都行。挑一个对你最方便的——所有这些工具都能加载本页推荐的开源 skill。如果你不知道选哪个，从 Claude Code 开始。

Claude Code
国外 · 推荐小白
# macOS brew install claude-code
Codex CLI
国外 · OpenAI
npm install -g @openai/codex
豆包 国内
Floatboat 国内
WorkBuddy 国内
用国内的 AI Agent？ 不用学命令行——把链接粘给它，它自己会装：
帮我把这个仓库装好，让我能用里面的所有 skill： https://github.com/coreyhaines31/marketingskills https://github.com/jinggreen15/ai-design-team https://github.com/op7418/Humanizer-zh
02
在本页找你要的 skill，点「复制」

本页上下所有代码块，右上角都有「复制」按钮。点一下，命令进剪贴板。skill 卡点开后里面的安装命令、prompt 模板，全都能一键复制。

# 比如装 marketingskills（39 个营销 skill 一锅端） git clone https://github.com/coreyhaines31/marketingskills \ ~/.claude/skills/marketingskills
→
已进剪贴板
03
粘进 AI Agent，再喊具体的小 skill

把命令粘进 AI Agent，它会自己 git clone、把 skill 装到正确位置。但要注意：装完仓库不等于自动会用。 你得明确喊出要用哪个小 skill——AI 才会去查那个 skill 的 SKILL.md 按指引干活。

# 装完后，第一句这样喊 — 明确点名要用的 skill 用 brief-analyst 拆这份业配需求， 再用 content-planner 给我 3 个切角， 最后用 humanizer-zh 过一遍去 AI 味输出。
💡 看到 AI 按 "brief-analyst → content-planner → humanizer-zh" 顺序切换工作风格，就说明它正确加载了 skill。 不喊名字，AI 只会用通用模式回答，那 skill 就白装了。
补充说明 · Skill 之外还需要什么

Skill 负责思路和流程，
生图 / 生视频 / 发布要靠外部工具。

Skill 教 AI 怎么想——比如 "用 image 写一个生图 prompt"。 但要真出图，需要接 ChatGPT Image 2.0 这种外部模型。要发飞书/小红书，需要接对应 MCP。

装完 skill 之后，你还要做两件事： ① 给 AI Agent 配上你常用的外部模型 API key ② 装好对应的 MCP server（飞书 / 小红书等）

生图模型
ChatGPT Image 2.0 · Midjourney · Sora-Image · 即梦
生视频模型
Seedance 2.0 · Sora · Runway · 即梦视频
外部数据 / MCP
飞书 CLI · 小红书 MCP · Notion API · Google APIs
完整配置 AI Agent（豆包/Claude Code/...） + Skill 包（这页 34 个 Skill） + 外部模型（生图生视频） + MCP（飞书/小红书） = 真正能跑的自动化工作流
